Google Integration

This app uses Google Sheets as the shared database (accessible from all phones) and Google Drive for file storage. Enter the Sheets ID and Drive Folder ID below.

📌 See setup guide at the bottom of this page for how to get these values.

Setup Guide

1. Go to sheets.google.com and create a new spreadsheet named "DocMgmtSystem"
2. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
3. Go to drive.google.com, create a folder named "DocMgmt Files"
4. Open the folder and copy the ID from the URL
5. Go to console.cloud.google.com → Enable Sheets API + Drive API → Create API Key
6. Share your Sheet and Drive folder with "anyone with link can edit"

✅ All phones using this app will then share the same live database automatically.
